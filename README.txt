"Программа" Телефонный справочник (ТС)

ТС была создана для хранения записей телефонных номеров пользователя (рабочего и сотового) с указанием ФИО и
наименованием организации.

Она позволяет смотреть список существующих контактов, вносить новые данные, редактировать и удалять уже имеющиеся данные,
а так же производить поиск по Фамилии, Имени, Отчеству или Организации.

Чтобы начать пользоваться программой нужно ввести следующие цифры:

  1 - для вывода всего списка справочника
  2 - для добавления новой записи в справочник
  3 - для редактирования записей в справочнике
  4 - для поиска записей в справочнике
  5 - для удаления контакта из справочника
